# ⚛️ React Counter App

A simple counter application built using **React** and **Vite**, deployed on **Netlify**.

## 🚀 Features

- Increment and Decrement counter
- Built with Vite for fast development and build
- Fully responsive and lightweight
- Deployed on Netlify
